# text-to-bg
For linux, takes a text file and converts it into a bg.

make sure you have magick and swaybg installed

make a new dir for the program to stay

put both the text file and the lua script in the new dir, the png is not needed

then go into the script with nvim nano etc.

change the config vars to your pref, note that pointsize, and font is in the magick command var

and if you are changing the font you have to list the address or see if magick is comp with it

change the text in the bg.txt file to the text you want your background to be

then ~/YOURDIR/bg-script.lua

if your unhapppy with the bg just pkill swaybg

and try again
